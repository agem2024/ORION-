# 🚀 SUBIR ORION A GITHUB

El repositorio local está **LISTO y LIMPIO**.
Todos los archivos (Manual, Landing, Código) ya están guardados (committed).

Solo necesitas ejecutar estos 3 comandos para subirlo a tu GitHub:

## 1. Crear Repositorio
Ve a [github.com/new](https://github.com/new) y crea un repo llamado `orion-clean`.

## 2. Conectar y Subir (Ejecuta esto en la terminal)
Copia y pega esto:

```powershell
# 1. Agrega tu GitHub (CAMBIA 'TU_USUARIO' por tu nombre real de GitHub)
git remote add origin https://github.com/TU_USUARIO/orion-clean.git

# 2. Sube los archivos
git branch -M main
git push -u origin main
```
